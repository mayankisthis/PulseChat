# PulseChat

Real-time chat platform built using the MERN stack and Socket.IO.

## Tech Stack

**Client:** React JS

**Server:** Node JS, Express JS

**Database:** Mongo DB
  

## Live Demo

Frontend: https://pulse-chat-nine-kohl.vercel.app
Backend: https://pulsechat-backend-7t1s.onrender.com

## Run Locally

Clone the project

```bash
  git clone https://github.com/mayankisthis/PulseChat.git
```

Go to the project directory

```bash
  cd PulseChat
```

Install dependencies

```bash
  npm install
```

```bash
  cd frontend/
  npm install
```

Start the server

```bash
  npm run start
```

Start the Client

```bash
  //open new terminal
  cd frontend
  npm start
```

# Features

- Real-time messaging
- Group chats
- Typing indicators
- Notifications
- JWT Authentication
- MongoDB persistence
